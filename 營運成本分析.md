# 網站營運成本分析（Google AdSense 廣告掛載）

## 📊 基本假設條件

### 用戶使用情況
- **日活躍用戶（DAU）：** 500人
- **每人每日問題數：** 3個問題
- **每日總問題數：** 500 × 3 = **1,500個問題**

### 廣告配置
- **廣告位數量：** 2個固定Google AdSense廣告位
- **用戶行為：** 不主動點擊廣告（僅展示）
- **廣告展示次數：** 每個問題頁面展示2個廣告 = 1,500 × 2 = **3,000次/天**

---

## 💰 成本計算

### 1. AI API 調用成本（OpenAI GPT-4 Turbo）

#### API 定價（參考 OpenAI 官方）
- **輸入（Prompt）：** $10 / 1M tokens
- **輸出（Response）：** $30 / 1M tokens

#### Token 使用估算
根據優化後的 prompt 設定：
- **每個問題的輸入（Prompt + System Prompt）：** 約 2,500 tokens
- **每個問題的輸出（Response）：** 約 2,000 tokens（平均，max_tokens=4000但不會全部用完）
- **每次調用總計：** 約 4,500 tokens

#### 每日成本計算
```
每日問題數：1,500個
每次調用 tokens：4,500
每日總 tokens：1,500 × 4,500 = 6,750,000 tokens

輸入成本：
- 輸入 tokens：1,500 × 2,500 = 3,750,000 tokens
- 輸入成本：3,750,000 / 1,000,000 × $10 = $37.50

輸出成本：
- 輸出 tokens：1,500 × 2,000 = 3,000,000 tokens
- 輸出成本：3,000,000 / 1,000,000 × $30 = $90.00

每日總成本：$37.50 + $90.00 = $127.50
```

#### 每月成本
```
每日成本：$127.50
每月成本：$127.50 × 30 = $3,825
台幣成本（匯率 1 USD = 32 TWD）：$3,825 × 32 = NT$ 122,400
```

---

### 2. 廣告收入（Google AdSense - 僅展示）

#### CPM（每千次展示費用）估算
- **台灣地區平均 CPM：** $0.50 - $2.00（約 NT$ 16 - NT$ 64）
- **中文內容 CPM：** 約 $0.80 - $1.50（約 NT$ 25.6 - NT$ 48）
- **保守估計（取中值）：** $1.00 / 1000次展示（約 NT$ 32）

#### 每日廣告收入
```
每日廣告展示：3,000次
CPM：$1.00 / 1000次
每日收入：3,000 / 1,000 × $1.00 = $3.00
台幣收入：$3.00 × 32 = NT$ 96
```

#### 每月廣告收入
```
每日收入：$3.00
每月收入：$3.00 × 30 = $90
台幣收入：NT$ 96 × 30 = NT$ 2,880
```

---

## 📈 淨成本計算

### 每月淨成本
```
AI API 成本：NT$ 122,400
廣告收入：NT$ 2,880
───────
淨成本：NT$ 122,400 - NT$ 2,880 = NT$ 119,520
```

### 每日淨成本
```
每日淨成本：NT$ 119,520 / 30 = NT$ 3,984
```

---

## 📊 成本結構分析

### 成本占比
- **AI API 成本：** 100%
- **廣告收入抵消：** 2.35%
- **淨成本：** 97.65%

### 每個問題的實際成本
```
每個問題的AI成本：NT$ 122,400 / (500 × 3 × 30) = NT$ 2.72
每個問題的廣告收入：NT$ 2,880 / (500 × 3 × 30) = NT$ 0.064
每個問題的淨成本：NT$ 2.72 - NT$ 0.064 = NT$ 2.656
```

---

## ⚠️ 注意事項與風險

### 1. 成本變動因素
- **Token 使用量可能更高：** 如果回答更詳細，可能超過估算
- **API 價格變動：** OpenAI 可能調整價格
- **匯率波動：** 台幣匯率變化影響成本

### 2. 收入變動因素
- **CPM 波動：** 廣告 CPM 會根據季節、行業、競爭變化
- **廣告填充率：** 不是每次都能展示廣告（可能只有 80-90%）
- **廣告攔截器：** 部分用戶使用廣告攔截器，減少展示

### 3. 實際成本可能更高
- **其他服務成本：** 伺服器、CDN、域名等
- **開發維護成本：** 人力成本
- **廣告收入可能被高估：** 實際 CPM 可能更低

---

## 💡 優化建議

### 1. 降低 AI 成本
- **使用更便宜的模型：** GPT-3.5-turbo（成本約為 GPT-4 的 1/10）
- **優化 Prompt：** 減少不必要的 tokens
- **緩存常見問題：** 對相似問題使用緩存回答
- **限制回答長度：** 降低 max_tokens（但會影響質量）

**使用 GPT-3.5-turbo 的成本估算：**
```
輸入：$0.50 / 1M tokens
輸出：$1.50 / 1M tokens

每日成本：
- 輸入：3,750,000 / 1,000,000 × $0.50 = $1.875
- 輸出：3,000,000 / 1,000,000 × $1.50 = $4.50
- 每日總計：$6.375
- 每月：$6.375 × 30 = $191.25
- 台幣：NT$ 6,120

淨成本：NT$ 6,120 - NT$ 2,880 = NT$ 3,240（節省 97%）
```

### 2. 增加廣告收入
- **增加廣告位：** 在結果頁面添加更多廣告
- **優化廣告位置：** 提高廣告可見性和點擊率
- **使用原生廣告：** 更自然的廣告展示
- **增加優質內容：** 提高 CPM

### 3. 混合策略
- **免費問題用 GPT-3.5：** 降低基礎成本
- **付費升級用 GPT-4：** 高質量回答收費
- **廣告 + 付費訂閱：** 雙重收入模式

## 📋 總結

### 當前配置（GPT-4 Turbo + 2個廣告位）
- **每月淨成本：** 約 **NT$ 119,520**
- **每日淨成本：** 約 **NT$ 3,984**
- **每個問題淨成本：** 約 **NT$ 2.66**

### 關鍵發現
1. ⚠️ **廣告收入僅抵消 2.35% 的成本**
2. ⚠️ **主要成本來自 AI API 調用（97.65%）**
3. ⚠️ **不點擊廣告的情況下，收入非常有限**

### 建議
1. 🔴 **優先考慮降低 AI 成本**（使用 GPT-3.5 或混合策略）
2. 🟡 **優化廣告策略**（提高點擊率或增加廣告位）
3. 🟢 **考慮付費訂閱模式**（補充收入來源）

### 如果改用 GPT-3.5 Turbo
- **每月淨成本：** 約 **NT$ 3,240**
- **節省：** **NT$ 116,280**（97% 成本降低）
- **質量影響：** 回答質量可能略有下降，但對於大多數問題仍足夠
